The folder structure should look like:

```text
mmagic
├── mmagic
├── tools
├── configs
├── data
|   ├── DPDD
|   |   ├── inputL
|   |   ├── inputR
|   |   ├── inputC
|   |   ├── target
|   |   ├── indoor_labels.npy
|   |   ├── indoor_labels.txt
|   |   ├── outdoor_labels.npy
|   |   ├── outdoor_labels.txt
```
